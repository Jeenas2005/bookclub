This page demonstrates how different mouse events work in JavaScript using buttons. Try clicking, double-clicking, hovering, and right-clicking below!

Clicking the button triggers a simple "click" event.

Double-clicking activates a different event than a single click.

Right-clicking is detected using the contextmenu event.

Mouse hovering triggers mouseover and mouseout events.

Pressing and releasing the mouse activates mousedown and mouseup.
